# perso-todo

페르소나 기반 개인 할일 관리 데스크탑 앱

## 다운로드

👉 **[최신 버전 다운로드](https://github.com/kiwonbyun/perso-todo/releases/latest)**

| 운영체제 | 파일 |
|---------|------|
| Mac (M1/M2/M3) | `perso-todo-x.x.x-arm64.dmg` |
| Mac (Intel) | `perso-todo-x.x.x.dmg` |
| Windows | `perso-todo Setup x.x.x.exe` |

## 기능

- 오늘 / 이번 주 / 월간 뷰
- 페르소나별 할일 분류 및 필터
- 할일 완료 시 폭죽 이펙트
- 메모 URL 자동 링크 + OG 미리보기
- 한국 공휴일 표시
- 데스크탑 알림 (원하는 시간 다중 설정)
- Slack Webhook 알림 연동
